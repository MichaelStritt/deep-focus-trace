# DeepFocus Trace

A minimalist focus and productivity tracker built with React, Vite, Tailwind CSS v4, and daisyUI v5.

# Development
```bash
npm install
npm run dev
```

# Build

```bash
npm run build
```

## Tech Stack

* **Framework:** React
* **Build Tool:** Vite
* **Styling:** Tailwind CSS v4
* **UI Components:** daisyUI v5
